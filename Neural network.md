# 📊 Neural Network Types

Now important part.

Neural networks ku multiple types irukku.

---

# 1️⃣ ANN (Artificial Neural Network)

👉 Basic neural network
👉 Fully connected layers

Use case:

* Tabular data
* Simple classification
* Basic regression

Example:

* Loan approval
* Customer churn

---

# 2️⃣ CNN (Convolutional Neural Network)

👉 Image-ku use pannuvanga
👉 Convolution layer use pannum

Why?
Image la:

* Local patterns important
* Edge detection
* Feature maps

Use case:

* Face recognition
* Medical image analysis
* Object detection

---

# 3️⃣ RNN (Recurrent Neural Network)

👉 Sequence data-ku use pannuvanga

Why?
Sequence la order important.

Example:
Sentence:
"I love AI"

Word order change panninaa meaning change aagum.

Use case:

* Text prediction
* Speech recognition
* Time series

Problem:

* Vanishing gradient
* Long memory issue

---

# 4️⃣ LSTM / GRU

👉 Advanced RNN version
👉 Long sequence handle pannum

Use case:

* Chatbots (old gen)
* Language modeling

---

# 5️⃣ Transformers (Modern Revolution)

🔥 Most important

RNN replace pannathu.

Use case:

* GPT
* BERT
* LLaMA
* Claude
* Gemini

Why powerful?

* Parallel processing
* Attention mechanism
* Large context handle pannum

ChatGPT = Transformer-based model.

# 🏗 Industry Perspective

| Problem Type          | Model Type  |

---

# 🧠 Why Not Use Only One Type?

Because:

Different data → different structure.

Image la:

* Pixel spatial relation important
  → CNN best

Text la:

* Word relation important
  → Transformer best

Tabular data la:
→ ANN sufficient
